Arena PLM for Google Docs

Version 2.2.0
Created by Daniel Bacon
This add-on was created to streamline document generation and PLM data integration for Arena users.
⚠️ Important: This add-on is provided free of charge with no official support. Use at your own discretion. For questions or to connect, reach out on LinkedIn.
What This Add-On Does
Seamlessly integrate PTC Arena PLM data into your Google Docs. Create document templates with Arena field tokens, generate documents with AI, and automatically populate them with live PLM data.
Key Features
✓ AI Document Generation
✓ Token Autodetect
✓ Token Palette
✓ Arena Search
✓ Smart Caching
✓ Rate Limiting
✓ Document Revisioning
✓ Real-time Filtering
